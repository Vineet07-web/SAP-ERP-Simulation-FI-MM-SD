# SAP ERP Simulation (FI-MM-SD)

## Project Description
This project simulates SAP ERP modules:
- FI (Financial Accounting)
- MM (Materials Management)
- SD (Sales and Distribution)

## Features
- Purchase Order (MM)
- Sales Order (SD)
- Inventory Management
- Financial Records (FI)

## Technologies Used
- Java
- Swing GUI

## How to Run
1. Compile:
   javac SAP_Advanced_GUI.java
2. Run:
   java SAP_Advanced_GUI
